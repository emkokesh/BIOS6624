# BIOS6624

This repository contains my project work for BIOS6624.

## Repository organization
This course includes multiple projects (0–4). This repository is organized as one folder per project, with subfolders for code, data (stored locally and ignored by git), and reports/results. 

## Projects
- **Project0 (archived):** `Project0/` – SPIT booklet timing agreement, adherence, and diurnal hormones.
- **Project1 (current):** `Project1/` – MACS HAART longitudinal analysis (baseline hard drug use and 2-year response).

## Folder structure for each project

| Folder | Description |
|---|---|
| `Background/` | contains the background information for the analysis |
| `Code/` | contains all R scripts / R Markdown files for this project |
| `DataRaw/` | contains all raw data provided by investigators (stored locally) |
| `DataProcessed/` | contains the processed data used for analysis (stored locally) |
| `Reports/` | contains all output, markdown files, figures, and the final report |

## Data policy
Data files are not uploaded to GitHub. They are stored locally and excluded using `.gitignore`.
